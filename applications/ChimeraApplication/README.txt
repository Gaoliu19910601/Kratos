To Compile the Chimera Applicaiton

    1. cd kratos/applications 
    2. inside CMakeLists.txt  -- Add the following

        if(${CHIMERA_APPLICATION} MATCHES ON)
          add_subdirectory(ChimeraApplication)
        endif(${CHIMERA_APPLICATIONN} MATCHES ON)

    3. Modify applications_interface.py accordingly

    4. cd kratos/cmake_build
    5. to configure.sh add 
        -DCHIMERA_APPLICATION=ON
